VRML (Virtual Reality Modeling Language)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Exports the PCB as a 3D model (WRL file).
This is intended for rendering, unlike STEP which is intended to be
an exact mechanic model

Type: ``vrml``

Category: **PCB/3D**
